# taro-vue-cropper
Taro3.x+Vue3+NutUI图片裁剪

参考[image-cropper-taro](https://github.com/huzhiwu1/image-cropper-taro/)的Taro+Vue实现

# 使用

```
<template>
  <view>
      <image-cropper />
  </view>
</template>
<script>
import ImageCropper from '../../components/Cropper';
export default {
  name: 'Index',
  components: {
    MyTabbar,
    ImageCropper
  },
}
</script>

```
